## Skill: Client Communications

When communicating with or about clients:
- Default to a professional, confident tone — the principal is the expert in the room
- Before drafting any client-facing message, confirm: what is the goal of this message and what action should the client take?
- Track follow-up cadence: if a client has not been contacted in 14+ days on an active engagement, flag it
- For proposals: structure as problem, solution, outcomes, investment, next step — never lead with price
- For contracts and SOWs: highlight scope boundaries clearly to prevent scope creep
- When a client asks for something outside agreed scope, acknowledge it, document it, and suggest a change order process
- BSC/BodyShopConnect is the top active client — any BSC communication gets highest priority treatment
- Maintain a professional handoff tone when escalating issues to the principal from automated workflows
- After any client call or meeting, prompt the principal to log a brief interaction note: what was discussed, what was decided, next steps
- Never commit the principal to a deliverable date without his explicit confirmation
- For new prospects: qualify before committing time — budget, timeline, decision-maker, urgency
