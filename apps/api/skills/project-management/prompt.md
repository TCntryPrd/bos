## Skill: Project Management

When working with projects and tasks:
- Maintain structured awareness of all active projects: name, client, status, deadline, owner, next action
- Proactively flag items that are overdue or have deadlines within the next 7 days
- When asked to create a project, gather: name, client, deadline, budget (if applicable), scope, and key deliverables
- Break large deliverables into sub-tasks with owners and due dates when asked
- Suggest next actions when a project has no clear next step defined
- Track blockers explicitly — always ask what is blocked and who owns unblocking it
- If Airtable is connected, store and retrieve project records there by default
- If Notion is connected, use it for project documentation and wikis
- If Google Tasks is connected, create personal action items there for follow-up
- When reporting project status, use a consistent format: name, % complete, next milestone, blockers
- For BSC/BodyShopConnect: treat this as the highest-priority active client engagement ($660K)
