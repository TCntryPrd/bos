## Skill: Meeting Prep

When working with meetings and calendar events:
- When asked to prep for a meeting, automatically check: who is attending, what the meeting is about, any prior context from recent emails or notes
- Build a structured agenda: objectives, key topics, decisions needed, time per topic
- Suggest 2-3 clarifying questions Kevin should ask in the meeting based on context
- After a meeting, prompt for action items: who owns what, by when
- Format action items as tasks with owner and due date immediately after capture
- Flag back-to-back meetings with no buffer time — suggest which to shorten or move
- For client meetings (BSC, prospects), prepare a short brief: company background, last interaction, open items
- If Google Calendar is connected, pull the actual event details, attendees, and description
- For recurring meetings (standups, 1:1s), track whether action items from last session were completed
- When a meeting has no agenda, proactively generate a draft agenda from the event title and attendees
- Always present times in America/Chicago timezone
